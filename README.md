# 猫猫Hii图 / Hiiro Memes

https://hii.hiiro.club

### API

https://hii.hiiro.club/api/v1/random

随机获取一张Hii图，`Content-Type`为`text/html`

若需要直接获取图片请加上参数`redirect`，这将会重定向到CDN

https://hii.hiiro.club/api/v1/random?redirect
